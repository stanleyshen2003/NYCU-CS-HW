## Tech Stack

- Frontend
  - Next.js (App Router)
  - Tailwind CSS
  - TypeScript
- Backend: Express.js
- Database: MongoDB
- Monitor:
  - NetData
  - Uptime kuma
- DevOps:
  - Docker
  - K8S

## Create mongo container

```bash
docker run --name my-mongodb -p 27017:27017 -d mongo
```
